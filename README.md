# **THIS PROGRAM CONTAINS FLASHING LIGHTS**

Something happens when the timer reaches 12 hours...

In order for the timer to work you need to manually edit the Date_And_Time.txt file in the zip folder.  
Don't delete the headers above the dates like StartingPoing_DateAndTime and Deadline_DateAndTime. It will not run without them  
The date and time formats are: DD.MM.YYYY and HH.MM.SS, they should specifically be in seprate lines and below the headers. Header should go first, then the date, then the time  
This file should be in the same directory as the .exe file
